# TypeScript CRUD API

## How to install dependencies
```
npm install
```

## How to start MySQL (macOS/Homebrew)
```
brew services start mysql
```
Or, if you installed MySQL another way:
```
mysql.server start
```

## How to stop MySQL
```
brew services stop mysql
```
Or:
```
mysql.server stop
```

## How to run the development server
```
npm run start:dev
```

## How to build for production
```
npm run build
```

## How to test the API
Use Postman or curl to send requests to your API endpoints (see your project handout for examples).
